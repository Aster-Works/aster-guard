## dev.to intro / English

MCP is powerful because it lets an AI coding tool reach files, commands, APIs, databases, and other developer systems. That also means MCP configuration deserves a quick safety check before it is connected.

Aster Guard MCP is a lightweight, local-first scanner for MCP and Claude Code configuration files. It does not start the MCP servers it scans. It reads the config, highlights risky patterns, and gives you a clear report before you connect.
